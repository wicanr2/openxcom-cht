------------------------------------------------------------------------------

  ⚠️ 本版本內僅含 X-COM 1 (UFO Defense) 遊戲資料。
  不要切換 mod（已知會 crash）。
  若要玩 TFTD 二代，請下載對應的 TFTD portable 版本。

 v2.20 特色
------------------------------------------------------------------------------

  - common UI 102%、xcom1 (UFO Defense) 102% 翻譯覆蓋率
  - 5 個 source patches (Font / Options / Geoscape clock / UFOpedia 色 x2 / 主選單簽名)
  - 6 國士兵名 CJK 音譯 (American/British/Chinese/Russian/German/French)
  - WQY Zen Hei Sharp 12px 點陣字型 (Apache 2.0 / GPL)
  - 主選單右下角 chibi pixel art 作者簽名

------------------------------------------------------------------------------
 系統需求
------------------------------------------------------------------------------

  - Windows 10 或 11 64-bit
  - 不必安裝 Visual C++ Redistributable (已內含 MSVCP140 / VCRUNTIME140)
  - 不必安裝 SDL (已內含 SDL / SDL_image / SDL_mixer / SDL_gfx 等 17 個 DLL)
