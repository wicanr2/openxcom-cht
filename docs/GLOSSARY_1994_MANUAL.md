## 二、翻譯策略

對照後採三種處理：

| 標籤 | 含義 |
|---|---|
| **沿用 1994** | 1994 譯名與本專案一致，繼續維持 |
| **歷史參考** | 1994 譯名屬時代用語，本專案採現代或社群慣例譯法，但於 UFOPEDIA 描述中可酌情引用 |
| **建議改回** | 1994 譯名更貼合遊戲機制 / 更精準，後續版本可考慮回歸 |
| **修正錯譯** | 1994 譯名為誤譯（如 STINGRAY），本專案採正確譯名 |

## 四、值得追加的 1994 風味

下列譯名 **1994 手冊獨有**，雖未列入本專案，但可作為 UFOPEDIA 描述的潤色參考：

- **「運兵機」**（SKYRANGER）— 比「天空遊俠」更貼合機制；後續版本建議改名
- **「黃貂魚式飛彈」**（STINGRAY）— **誤譯修正**：STINGRAY 是黃貂魚（魟），「刺尾」為錯譯
- **「啟動引信」**（PRIME GRENADE）— 比「啟動」更技術正確
- **「醫藥箱」**（MEDI-KIT）— 比「醫療包」更貼近 KIT 原意
- **「不醒人事」**（UNCONSCIOUS）— 比「昏迷」更生動、更貼近 1994 中文遊戲風味

## 五、1994 手冊的限制

- 手冊僅列 **英文速查表**，不含完整翻譯（畫面字串、UFOPEDIA 描述等仍是英文原版）
- 未涵蓋的範圍：
  - 外星人 **種族名**（Sectoid、Floater、Muton 等 — 因英文版本身未在介面顯式命名）
  - **建築物描述**（DEFENSE_LASER、HYPERWAVE_DECODER 等晚期設施）
  - **UFOPEDIA 全文**（手冊只列 ACCESS、ARMAMENT 等欄位名）
  - **任務劇情**（briefings、cutscenes）
- 含 OCR / 排版錯字：BAZIL→BRAZIL、GROUNT→GROUND、TOUHG→TOUGH、PANICK→PANIC、UNCONCIOUS→UNCONSCIOUS、LAAUNCHER→LAUNCHER、SERGENT→SERGEANT
